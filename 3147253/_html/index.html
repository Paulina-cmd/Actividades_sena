Glosario Tecnológico

A - M

A | B | C | D | E | F | G | H | I | J | K | L | M N-Z

A

Algoritmo: Conjunto de instrucciones o reglas definidas para realizar una tarea.

B

Bug: Error o defecto en un software que causa un comportamiento inesperado.

C

Compilador: Programa que traduce el código fuente a un lenguaje ejecutable.

D

Depuración: Proceso de identificar y corregir errores en el código.

E

Etiquetas: Elementos de código HTML que definen la estructura de una página web.

F

Framework: Conjunto de herramientas y bibliotecas que facilitan el desarrollo de software.

G

Git: Sistema de control de versiones distribuido utilizado en el desarrollo de software.

H

HTML: Lenguaje de marcado utilizado para crear páginas web.

I

Intérprete: Programa que ejecuta código directamente, sin necesidad de compilarlo previamente.

J

JavaScript: Lenguaje de programación utilizado para crear interactividad en páginas web.

K

Kernel: Núcleo del sistema operativo que gestiona los recursos del hardware.

L

Librería: Conjunto de funciones y procedimientos que pueden ser utilizados por un programa.

M

Markdown: Lenguaje de marcado ligero utilizado para formatear texto.
