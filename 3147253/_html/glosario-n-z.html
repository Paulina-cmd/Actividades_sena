Glosario Tecnológico

N - Z

← Volver a A-M

N

Netbook: Computadora portátil pequeña y económica.

O

OAuth: Protocolo de autorización que permite el acceso a servicios sin compartir credenciales.

P

Python: Lenguaje de programación de alto nivel y fácil de aprender.

Q

Quicksort: Algoritmo eficiente de ordenación basado en el principio de divide y vencerás.

R

Router: Dispositivo que enruta el tráfico de datos entre redes de computadoras.

S

SQL: Lenguaje de consulta estructurado utilizado para interactuar con bases de datos.

T

Token: Unidad de autenticación utilizada para acceder a un servicio de manera segura.

U

UDP: Protocolo de comunicación sin conexión utilizado en redes.

V

VPN: Red privada virtual utilizada para acceder de manera segura a una red remota.

W

Wi-Fi: Tecnología de comunicación inalámbrica utilizada para redes locales.

X

XHTML: Extensión de HTML que utiliza una sintaxis más estricta basada en XML.

Y

YAML: Lenguaje de marcado utilizado para representar datos estructurados de manera legible.

Z

Zero-Day: Vulnerabilidad de seguridad desconocida en el software antes de ser parcheada.
